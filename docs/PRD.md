# Product Requirements Document

## Product Overview

Stock Analysis is a stock portfolio tracker that helps users understand what they own, what they bought or sold, and how their portfolio value changes over time.

The first version is focused on reliable portfolio tracking. Later versions will add portfolio analysis, insights, and deeper decision-support features.

## Why This Project Exists

Investors often need a simple place to track holdings, transactions, and portfolio performance without paying for expensive market data tools. This project should provide a practical, personal portfolio tracker that is free to develop and can grow into a more advanced analysis platform over time.

## MVP Goals

The MVP should let a user:

- Create one or more portfolios.
- Add buy and sell transactions.
- Search for stock tickers.
- View current or delayed market prices for tracked tickers.
- See portfolio performance charts for `1d`, `5d`, `1m`, `6m`, `1y`, and `5y`.
- Use the product without requiring paid APIs or paid development services.

## User Experience Goals

- Make portfolio setup quick and understandable.
- Make transaction entry clear enough that users can trust their holdings.
- Show the current portfolio value and performance without clutter.
- Keep the interface ready for future analysis features without making the MVP feel unfinished.

## Success Criteria

- A user can create a portfolio and add buy/sell transactions.
- The app can calculate holdings from transaction history.
- The app can look up tickers and show current or delayed prices.
- The app can display portfolio value over the supported time ranges.
- The MVP remains free to develop.

## Out of Scope for Initial MVP

- Paid real-time market data.
- Brokerage account linking.
- Tax reporting.
- Options, crypto, forex, or other non-stock asset classes unless added later.
- Advanced analysis, recommendations, or automated trading.
